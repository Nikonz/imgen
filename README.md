# imgen
Script for generating images with text.

![](examples/021.png)
![](examples/033.png)
![](examples/099.png)

## How to run

### Examples

Generate 100 images:
```
./imgen.py 100
```
Generate 100 images with existing words and the specified image and font sizes.

```
./imgen.py 100 --dtype words --image_size 512 512 --font_size 90
```

### Further help

```
./imgen.py -h
```
